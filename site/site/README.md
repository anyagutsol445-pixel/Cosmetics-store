# 📦 Project Setup Guide (XAMPP)

## 📋 Вимоги

Перед початком переконайся, що в тебе встановлено:

* [XAMPP](https://www.apachefriends.org/index.html) (рекомендовано останню версію)
* Браузер (Chrome / Firefox)
* Git (опціонально)

---

## 📥 Встановлення проєкту

### 1. Завантаження проєкту

**Варіант 1 (ZIP):**

* Розпакуй архів у папку:

```
C:\xampp\htdocs\
```

**Варіант 2 (Git):**

```bash
cd C:\xampp\htdocs\
git clone <your-repository-link>
```

---

### 2. Структура проєкту

Переконайся, що структура виглядає приблизно так:

```
C:\xampp\htdocs\your-project\
    ├── index.php
    ├── config.php
    ├── assets/
    ├── includes/
    └── ...
```

---

## ⚙️ Налаштування XAMPP

1. Запусти **XAMPP Control Panel**
2. Увімкни:

   * ✅ Apache
   * ✅ MySQL

---

## 🗄️ Налаштування бази даних

1. Перейди в браузері:

```
http://localhost/phpmyadmin
```

2. Створи нову базу даних (наприклад):

```
my_database
```

3. Імпортуй SQL-файл:

* Натисни **Import**
* Вибери файл:

```
database.sql
```

* Натисни **Go**

---

## 🔧 Конфігурація підключення до БД

Відкрий файл:

```
config.php
```

І зміни налаштування:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "my_database";
```

> ⚠️ За замовчуванням у XAMPP пароль порожній

---

## 🚀 Запуск проєкту

Відкрий у браузері:

```
http://localhost/your-project/
```

---

## ❗ Можливі проблеми

### 🔴 Apache не запускається

* Перевір порт 80 (може бути зайнятий Skype / іншим софтом)
* Спробуй змінити порт у `httpd.conf`

---

### 🔴 MySQL не запускається

* Перевір, чи немає конфлікту з іншими сервісами (наприклад, MySQL Server)

---

### 🔴 Помилка підключення до БД

* Перевір `config.php`
* Переконайся, що база даних існує

---

## 📁 Додатково

* Всі файли повинні бути в `htdocs`
* PHP версія в XAMPP має відповідати вимогам проєкту
